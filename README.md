This project will demonstrate on how to receive binary packet of fixed length. 
   * The code will validate the packet with a calculated checksum which should match the last byte of the packet.
   * The code will ignore packets that are short in bytes until a set amount of bytes are received.   

See the Wiki for more information https://github.com/karlyamashita/NUCLEO-G071RB_ParseTelemetryData/wiki
